¿Qué es un framework?

Un framework es un conjunto estaandarizaado de conceptos, prácticas y criterios para enfocr un tipo de problemática particulaar que sirve como referenciaa, paara enfrentar y resolver nuevos problemas de índole similar.

A continuación veremos algunos de los frameworks que utilizan Node.js más usados.

  • Express.js

    Express.js es un framework que actua como middleware: ayuda a establecer y configurar rutas para enviar y recibir peticiones entre el front-end y el back-end.

    Es muy ligero y provee al desarollador de una variedad de herramientas para servidores HTTP, es muy bueno para desarollar paginas web, o incluso HTTP APIs. Soporta una variedad de motores de plantillas para facilitar el desarollo de apps web.

  • Socket.IO

    Socket.IO es una liberira de Javascript utilizada para construir apps en "tiempo real" y establecer una comunicación bidireccional entre el cliente y el servidor

    Este framework es ideal para desarrollar aplicaciones que necesitan estar desarrolladas con websockets.

  • Meteor.js

    Meteor es un framework de javascript full-stack que ayuda a desarrolladores a crear applicaciones web en tiempo real. Funciona de tal manera que el servidor envia datos en vez del documento html y el cliente renderiza esta información

  • Sails.js

    Es un framework MVC (Modelo Vista Controlador) para node que esta desarrollado sobre express. Soporta APIs REST autogeneradas con una sencilla integración de WebSocket.

    Es compatible con la mayoria de bases de datos populares, ya esan sql o no-sql. Y a su vez tambien es compatible con la mayoria de tecnologias para el front-end, como Angular y parecidos.

  • Nest.js

    Esta desarrollado en TypeScript, lo que siginifica que tiene un tipado fuerte. Combina elementos de la programación orientada a objetos, programación funcional y programación funcional reactiva.

    Nest sirve como un paquete completo de desarrollo para desarrollar aplicaciones profesionales, ademas utiliza Express, pero es compatible con un amplio rango de librerias.
