Node.js

¿Que es Node.js?

Node.js es un entorno en tiempo de ejecución multiplataforma, de codigo abierto, para la capa del servidor (pero no se limita a ello) basado en el lenguaje de programación JavaScript, asíncrono, con E/S de datos en una arquitectura orientada a eventos y basado en el motor V8 de Google. Fue credo con el enfoque de ser útil en la creación de programas de red altamente escalales, como por ejemplo servidores web. Fue creado por Ryan Dahl en 2009 y su evolución está apadrinada por la empresa Joyent, que además tiene contratado a Dahl en plantilla.

Node.js es similar en su propósito a Twisted o Tornado de Python, Perl Object Environment de Perl, libevent o libev de C, EventMachine de Ruby, vibe.d de D y Java EE de Java existe Apache MINA, Netty, Akka, Vert.x, Grizly o Xsocket. AAL contrario que la mayoria del código JavaScript, no se ejecutaa en un navegador, sino en el servidor. Node.js implementa algunas especificaciones del CommonJS. Node.js incluye un entorno REPL para depuración interactiva.

¿Para que se utiliza Node.js?

Node.js principalmente se usa para servidores orientados a eventos, no bloqueantes, ya que solo tiene un hilo de ejecución. Se usa para páginas web normales y para servicios de API de backend.

Gestion de paquetes y modulos en node.

Para la gestión y mantenimiento de modulos Node.js utiliza una herramienta llamada NPM ( Node Package Manager ), con esta herramienta nosotros podemos descargar modulos para poder usarlos en nuestras aplicaciones o crear nosotros nuestros propios modulos y subirlos para poder utilizarlos en otros proyectos o que incluso otros desarrolladores los puedan usar.
